#### DATA:
##### Data is in form of image - chest xrays.
##### It contains 3 classes - Normal, Pneumonia and Covid-19.

#### DATA PREPROCESSING:
##### Data is preprocessed and loaded onto the data loader.

#### MODEL TRAINING AND EVALUATION:
##### Model is trained and performs with accuracy of 92%.

#### FLASK APP:
##### Flask app is used to get chest xray from user and result obtained from trained model is displayed on the web app.
